Merge Map

This operator maps each source value to an inner observable and merges the values from multiple inner observables into a single observable. It does not cancel or unsubscribe from any inner observables.

When to Use

You need to process multiple concurrent inner observables at the same time, without canceling any previous ones. You don't mind receiving results in an interleaved order (the order of values from inner observables might be mixed). You have independent async tasks or requests that can run in parallel.
