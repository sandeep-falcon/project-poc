From

The from function takes an array, iterable, promise, or any observable-like object and converts it into an observable that emits each item in the sequence or the resolved value in the case of promises.

When to Use from

Use from when you want to turn a collection (array) or a promise into an observable. This is common when you have data from APIs (promises) or multiple items in a collection.
